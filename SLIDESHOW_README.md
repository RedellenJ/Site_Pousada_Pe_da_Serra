# Slideshow da Página Inicial

Para o slideshow funcionar corretamente, adicione as seguintes imagens na pasta `Img/`:

- `slide1.jpg` - Primeira imagem do slideshow (ex: vista externa da pousada)
- `slide2.jpg` - Segunda imagem do slideshow (ex: quarto ou área comum)
- `slide3.jpg` - Terceira imagem do slideshow (ex: café da manhã ou natureza)
- `slide4.jpg` - Quarta imagem do slideshow (ex: área de lazer ou detalhes)

## Características do slideshow:
- Cada imagem fica visível por 5 segundos
- Transição suave entre as imagens
- Fundo com overlay verde para manter legibilidade do texto
- Imagens em full-screen no fundo da seção hero

## Dimensões recomendadas:
- Resolução: 1920x1080 ou superior
- Formato: JPG ou PNG
- Orientação: Paisagem (horizontal)

As imagens devem ser otimizadas para web para melhor performance.
